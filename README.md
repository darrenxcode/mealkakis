Introduction
Welcome to the mealkakise application setup guide. This document will walk you through the steps to clone the application and prepare your environment to run it.

Prerequisites
Before you begin, ensure that you have the following prerequisites installed on your system:

MealKakis-Backend:
Java Development Kit (JDK) 17 or higher
Maven build tool
Git version control system
InteliJ

MealKakis-Frontend:
NodeJS
VScode

If you do not have any of these prerequisites installed, please refer to their respective documentation for installation instructions.
